# 千分符

## 题目描述
将数字字符串转换为千分位格式（例如将 1234567.89 变为 1,234,567.89），其中第二个参数用于表示保留的小数位数。

## 思路分析
1. 千分符只存在整数部分，需要将负号、整数、小数拆分出来。
2. 以小数点为界，拆分为整数部分和小数部分。
3. 将整数部分 3 个为一组拆分。

具体实现上有：
1. 顺序遍历。核心逻辑是在遍历的时候判断剩余的字符长度是否是 3 的倍数，如果是就插入分隔符。
2. 逆序遍历。3 个为一组是从个位数开始的，所有可以从右往左遍历，维护一个 count，每当 count 是 3 的倍数就插入分隔符。
3. 正则表达式。使用正则将字符每 3 个为一组拆分。然后替换原字符的时候带上分割符。
4. 使用内置的 Intl API 实现。

```js
function formatIntl(numStr, precision) {
  // 注意：Intl 必须接收数字或 BigInt
  // 如果是超大浮点数，此方法会丢失精度
  const n = parseFloat(numStr); 
  
  return new Intl.NumberFormat('en-US', {
    minimumFractionDigits: precision,
    maximumFractionDigits: precision
  }).format(n);
}
```
